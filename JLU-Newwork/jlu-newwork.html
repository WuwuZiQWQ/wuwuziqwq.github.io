1.download dogcom and compile

git clone https://github.com/mchome/dogcom.git

$ make # Linux
$ make win32=y # Windows(MinGW)
$ make test=y # For testing purposes
$ make force_encrypt=y # Force open encrypt mode in PPPoE version

2.config dogcom.conf

you can use dogcom.example.conf to generate you profile

Then you should change the blank var in file, usually you should fill in your username,passward,ip and your device's MAC address.

If you finish changing the blank var in file, then change the file name to dogcom.conf and move it to the directory of dogcom

ps:If you are a student from JLU , the server in conf file is right.Other school's student shuld change it to your shool's server.

3.run dogcom

you can run dogcom by powershell

here is command: dogcom.exe -m dhcp -l path-to-dogcom/dogcom.log -c path-to-dogcom/dogcom.conf
